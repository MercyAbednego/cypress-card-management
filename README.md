# Cypress Card Management Automation

## Overview
This project automates card management workflows using Cypress.

The automation covers end-to-end testing of key card operations in a fintech application.

---

## Features Tested

✅ Valid Login  
✅ Invalid Login  
✅ Card Request Creation  
✅ Card Linking  
✅ Card Status Update  
✅ Account Information Verification

---

## Tools Used

- Cypress
- JavaScript
- VS Code
- Git & GitHub

---

## Test Flow

1. Login to application
2. Navigate to Card Requests
3. View card details
4. Create card request
5. Link card
6. Update card status
7. Verify account information
